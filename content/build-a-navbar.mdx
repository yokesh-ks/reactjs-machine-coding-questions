## Requirements

Build a responsive navigation bar that:
- Renders a horizontal nav with logo and navigation links on desktop
- Shows dropdown sub-menus on hover/click for links with children
- Collapses into a hamburger icon on screens below 768px
- Toggles a full-width mobile menu when the hamburger is clicked
- Closes dropdowns and the mobile menu when the user clicks outside
- Marks the active route with a highlighted style
