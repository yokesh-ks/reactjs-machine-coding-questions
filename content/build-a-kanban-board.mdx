## Requirements

Build a Kanban board with:
- Three columns: **To Do**, **In Progress**, and **Done**
- Tasks displayed as draggable cards in each column
- Ability to drag a task card and drop it into a different column
- Visual feedback (highlight) on the column being dragged over
- Ability to add a new task to any column via an inline input
- Ability to delete a task card from any column
